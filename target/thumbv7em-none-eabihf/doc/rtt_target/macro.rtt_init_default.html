Macro rtt_init_default

Expand description

Initializes RTT with default up/down channels.

The default channels are up channel 0 with a 1024 byte buffer and down channel 0 with a 16 byte buffer. Both channels are called “Terminal”. This macro is equivalent to:

rtt_init! {
    up: {
        0: {
            size: 1024
            name: "Terminal"
        }
    }
    down: {
        0: {
            size: 16
            name: "Terminal"
        }
    }
};

See rtt_init for more details.
