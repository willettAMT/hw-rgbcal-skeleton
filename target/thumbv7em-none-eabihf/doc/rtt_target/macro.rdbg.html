Macro rdbg

macro_rules! rdbg {
    (=> $terminal:expr) => { ... };
    (=> $terminal:expr, $val:expr $(,)?) => { ... };
    (=> $terminal:expr, $($val:expr),+ $(,)?) => { ... };
    () => { ... };
    ($val:expr $(,)?) => { ... };
    ($($val:expr),+ $(,)?) => { ... };
}
Expand description

Print to RTT and return the value of a given expression for quick debugging. This is equivalent to Rust’s std::dbg!() macro.
