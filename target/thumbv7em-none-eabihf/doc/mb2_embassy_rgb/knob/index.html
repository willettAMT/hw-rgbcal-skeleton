§Knob Input Module

This module provides analog input handling for potentiometer/knob controls using the nRF52’s SAADC (Successive Approximation ADC) peripheral.

The knob converts analog voltage readings into discrete levels suitable for controlling RGB LED intensity or other stepped parameters.`
